# 系統要求
1. Node.js 14.15
2. npm 6.14

# Install
1. ```npm i```
2. ```npm i -g apidoc gaze```

# Use
### Edit doc
1. build apidoc
```
npm run build
```
2. open ```doc/index.html``` with browser

### Auto Rebuild
1. watch apidoc
```
npm run dev
```
